# Учебный проект для лабораторных работ по дисциплине "Основы Web-технологий"

Учебная версия дизайна для сайта компании Персона

Приложения протестировано только для разрешения экрана 1920x1080

Использованные технологии:

* vite + vue
* tailwindcss
* primevue
* vueuse
* yandex карты

## Установка

Установка зависимостей:

```bash
# npm
npm install

# pnpm
pnpm install

# yarn
yarn install

# bun
bun install
```

## Сервер разработки

Запуска сервер на `http://localhost:3000`:

```bash
# npm
npm run dev

# pnpm
pnpm run dev

# yarn
yarn dev

# bun
bun run dev
```

## Сборка

Для сборки приложения:

```bash

# pnpm
pnpm run build

# yarn

# bun
bun run build
```
